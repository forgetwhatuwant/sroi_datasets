Dataset Overview

The SROI dataset contains demonstrations of expert strawberry picking operations, collected using a teleoperated robotic system. Each demonstration includes synchronized data from multiple sensors, capturing the complete picking process from approach to successful harvest.

Key benefits of the SROI dataset:

  • Multi-sensor Fusion: Combines stereo RGB images from OAK camera and Intel RealSense D435, along with IMU data for robust perception and motion analysis.
  • High-Quality Demonstrations: Expert demonstrations ensure high-quality training data for imitation learning and policy development.
  • Comprehensive Coverage: Includes various picking scenarios, lighting conditions, and strawberry cluster configurations for diverse training scenarios.
  • Time-Synchronized Data: All sensor data is precisely synchronized, enabling accurate temporal analysis and multi-modal learning.
  • Rich Annotations: Includes action status information and camera calibration data for precise analysis and system development.

Dataset Structure

The dataset is organized as ROS bags containing the following topics:

  • OAK Camera Topics:
    • /oak/left/image_raw/compressed - Left camera RGB images
    • /oak/right/image_raw/compressed - Right camera RGB images
    • /oak/imu/data - IMU sensor data
  • Intel RealSense D435 Topics:
    • /camera/color/image_raw/compressed - RGB camera images
    • /camera/infra1/image_rect_raw/compressed - Left IR camera images
    • /camera/infra2/image_rect_raw/compressed - Right IR camera images
    • /camera/imu - IMU sensor data
    • /camera/accel/imu_info - Accelerometer information
    • /camera/gyro/imu_info - Gyroscope information
    • /camera/color/camera_info - RGB camera calibration info
    • /camera/infra1/camera_info - Left IR camera calibration info
    • /camera/infra2/camera_info - Right IR camera calibration info
  • System Topics:
    • /upi/status/is_action - Action status information
    • /clock - ROS time synchronization


Visualizing the Dataset on the Web

You can directly visualize the ROS bag data in your web browser using Foxglove Studio, a powerful web-based visualization tool that allows for interactive exploration of your data without needing to install desktop software.

Using Foxglove Studio

Here's how to get started:

  1. Visit Foxglove Studio in your browser.
  2. Click "Open local file" and select your ROS bag file.
  3. Configure your layout by adding panels for:
    • Image panels for:
      • OAK camera: /oak/left/image_raw/compressed and /oak/right/image_raw/compressed
      • D435 camera: /camera/color/image_raw/compressed, /camera/infra1/image_rect_raw/compressed, and /camera/infra2/image_rect_raw/compressed
    • Plot panels for:
      • OAK IMU: /oak/imu/data
      • D435 IMU: /camera/imu
    • Raw Messages panel for /upi/status/is_action
  4. Use the playback controls to navigate through the data.


Dataset Usage

To help you get started with the SROI dataset, we provide a set of utilities for processing and analyzing the ROS bag files. These utilities include:

  • ROS bag processing scripts
  • Data extraction and conversion tools
  • Example visualization code
  • Utility functions for common tasks

Visit our SROI_rosbags_utilities repository to access these tools and get started with your analysis.
